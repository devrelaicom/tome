# Phase 5 Retrospective: User Story 3 — Onboard as a contributor

## What Worked Well

- **The scaffold was already correct.** `lefthook.yml`, `.github/workflows/ci.yml`,
  `.github/workflows/security.yml`, `LICENSE-MIT`, `LICENSE-APACHE`,
  `README.md`, and `CONTRIBUTING.md` all matched the audit checklist
  (T102, T105, T106, T108, T109) on first inspection. The repository URL
  was the only thing out of date — the scaffold pointed at
  `aaronbassett/tome` and the real remote is `devrelaicom/tome`. Fixed in
  README, CONTRIBUTING, manifest.rs's `SCHEMA_URI`, and `Cargo.toml`.
- **Conventional Commits via `cog verify`.** The `commit-msg` hook has
  fired on every commit this PR; every commit message in the log parses
  cleanly.
- **`pre-push` running `cargo test --workspace`.** Caught nothing yet (every
  push has been green locally first) but the cost is real — push from a
  cold cache is ~10 s longer because the test suite runs. Contributors
  with patience will appreciate that CI rarely surprises them.

## What Didn't Work

- **`CONTRIBUTING.md` referenced `CODE_OF_CONDUCT.md`** which the scaffold
  never produced. Two reasonable fixes: create a one-liner pointing at the
  Contributor Covenant text, or change the link to point at the upstream
  Covenant URL directly. Took the second option — keeps the project's
  surface area smaller and avoids drift between the local copy and the
  canonical text.

## Workarounds & Solutions

- Manual 10-minute on-ramp verification (T110) on this machine:
  cold-cache `cargo build` was 41 s; `cargo test` was 6 s. With
  `lefthook install` and one trivial commit, total time from `git clone`
  to "ready to push" is around two minutes. SC-002's 10-minute budget is
  comfortable.

## Packages & Dependencies

- No new crates this phase. `cog` and `lefthook` are still installed via
  `brew` / `cargo install`; documented in CONTRIBUTING.md.

## Patterns & Code

- **Repo URL as a single point of truth.** The next time this changes
  (likely a public-namespace rebrand) the touchpoints are: `README.md`,
  `CONTRIBUTING.md`, `Cargo.toml`'s `repository`, and `src/catalog/
  manifest.rs::SCHEMA_URI`. A `grep -r aaronbassett/tome` (or whatever
  the old org is) catches them all.

## For Next Time

- Consider extracting the schema URI into a constant set in `build.rs` from
  env at compile time, so the URL doesn't need to be updated by hand.
  Deferred — small payoff, not blocking anything.
