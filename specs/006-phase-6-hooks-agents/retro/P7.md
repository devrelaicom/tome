## What this phase landed

Plugin-agent privileged fields (`hooks`/`mcpServers`/`permissionMode`) pass
through to Claude Code by default but are auditable via doctor's
`PrivilegeEscalationReport` and strippable via the opt-in layered
`strip_plugin_agent_privileges` setting. `tome doctor` gains five emit-only
reports (Hooks/Guardrails/Agents/PrivilegeEscalation/Persona) with byte-stable
JSON pins; `--fix` repairs only the safe derivable cases (re-render stale
guardrails, re-emit missing agents, remove orphaned `<plugin>__*`) and never
removes a non-matching hook or user content. `tome plugin show` lists agents
+ hooks.json/GUARDRAILS.md presence + resolved persona name.
